## Project Overiew

A simple and interactive movie tracking app built with React. Users can search for movies, rate them, and maintain a personal watched list.

---

## Features

1. Search Movies
  - Search movies by title using OMDb API.
2. Movie Ratings
  - Rate movies using a star rating system.
  - Track your personal ratings for watched movies.
3. Watched List
  - Add movies to a personal watched list.
  - View all watched movies with ratings and details.
  - Summary of watched movies (average rating, total watched, etc.).
4. Movie Details
  - View detailed information about selected movies.
  - Includes title, year, poster, and user rating.
5. Persistent Storage
  - Watched movies and ratings are saved in localStorage.
  - Data persists across page reloads.
6. User-friendly UI
  - Interactive and responsive layout.
  - Clean, reusable components (Movie, WatchedMovie, StarRating, MoviesList, WatchedMoviesList, etc.).


  ---

  ## Tech Stack
  - React, Context API, React Hooks
  - State Management: useReducer + Context
  - Persistent Storage: LocalStorage
  - API: OMDb API for movie search
  - Styling: Plain CSS with reusable components
